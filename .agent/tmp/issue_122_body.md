## 📋 범위
- Google News RSS를 파싱해 `crawled_news`에 upsert 합니다.
- MVP 목표는 '완전 자동화'가 아니라 운영 부담을 낮추는 수준입니다.

## ✅ 완료 조건
- [ ] RSS 소스/키워드 목록 확정(운영 가능한 수준)
- [ ] URL 기준 중복 제거 + upsert
- [ ] 최신 N개 유지 정책(예: 200개)
- [ ] 실행 주기: 1일 1회(스케줄러/크론)
- [ ] 문서 갱신: `docs/mvp/mvp_크롤링.md`

- Airflow/Celery 대규모 파이프라인
