# 😆 FanPulse 화면 정의서

## 1. 개요

본 문서는 **FanPulse** 서비스의 주요 화면을 정의하며, 각 화면의 기능, 레이아웃, 인터랙션을 상세히 설명합니다.

---

## 2. 주요 화면 및 기능

### 2.1 메인 화면 (H001)

**설명**: K-POP 관련 최신 뉴스, 인기 게시글, 차트 순위를 표시하는 홈 화면

| 항목                  | 내용                                                                                                                                       |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| 화면 ID               | H001                                                                                                                                       |
| 화면명                | 메인 화면                                                                                                                                  |
| 주요 기능             | - 최신 K-POP 뉴스 표시<br>- 인기 게시글 리스트 제공<br>- 실시간 차트 순위 표시<br>- 각 기능으로 이동하는 네비게이션 제공                   |
| 주요 버튼 및 인터랙션 | - **뉴스 카드 클릭** → 뉴스 상세 페이지 (H011)<br>- **인기 게시글 클릭** → 팬 커뮤니티 (H003)<br>- **차트 클릭** → 차트 순위 페이지 (H005) |

---

### 2.2 로그인/회원가입 (H002)

**설명**: OAuth2 기반 SNS 로그인 및 신규 회원 가입을 지원하는 화면

| 항목                  | 내용                                                                                                         |
| --------------------- | ------------------------------------------------------------------------------------------------------------ |
| 화면 ID               | H002                                                                                                         |
| 화면명                | 로그인/회원가입                                                                                              |
| 주요 기능             | - Google, Kakao, Apple 계정 로그인 지원<br>- 신규 회원 가입 시 기본 프로필 설정                              |
| 주요 버튼 및 인터랙션 | - **SNS 로그인 버튼 클릭** → 인증 후 메인 화면 (H001)<br>- **회원가입 진행** → 가입 완료 후 메인 화면 (H001) |
